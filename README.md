# HOTEL RIZZ - Restaurant Website

A modern and responsive restaurant website built with HTML, CSS, and JavaScript, featuring a beautiful UI with smooth animations and interactive elements.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth scrolling navigation with mobile-friendly hamburger menu
- **Search Functionality**: Full-screen search overlay with elegant animations
- **Image Sliders**: 
  - Home slider showcasing special dishes using Swiper.js
  - Customer review slider with auto-play functionality
- **Product Sections**:
  - Popular Dishes showcase
  - Detailed Menu with pricing
  - Customer Reviews
- **Order Form**: Easy-to-use order form with multiple input fields
- **Icon Integration**: Font Awesome icons for enhanced visual appeal

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- [Swiper.js](https://swiperjs.com/) - For touch sliders
- [Font Awesome](https://fontawesome.com/) - For icons
- Google Fonts (Nunito)

## Sections

1. **Home**: Eye-catching slider featuring special dishes
2. **Dishes**: Grid display of popular dishes with prices and "Add to Cart" buttons
3. **About**: Information about the restaurant and its services
4. **Menu**: Detailed menu items with descriptions and ratings
5. **Review**: Customer testimonials in a slider format
6. **Order**: Online order form for quick ordering
7. **Footer**: Contact information and quick links

## Color Scheme

- Primary Green: `#27ae60`
- Dark Blue: `#192a56`
- Light Gray: `#666`
- Background: `#eee` (alternate sections)

## Responsive Breakpoints

- Mobile: 450px and below
- Tablet: 768px and below
- Desktop: 991px and above

## Setup & Installation

1. Clone or download the repository
2. Open `index.html` in your web browser
3. No additional dependencies or build process required
